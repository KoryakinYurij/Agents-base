# Репозиторий для разработки ИИ-агентов

Этот репозиторий предназначен для создания и тестирования ИИ-агентов, в первую очередь с использованием Google Agent Development Kit (ADK).

## Начало работы

### Необходимые инструменты

Перед началом работы убедитесь, что у вас установлены:

*   **Python (версия 3.9 или выше)**
*   **pip** (менеджер пакетов Python)
*   **virtualenv** (инструмент для создания изолированных окружений Python)

### Установка

1.  **Склонируйте репозиторий:**
    ```bash
    git clone <URL репозитория>
    cd <имя папки репозитория>
    ```

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    python -m venv .venv
    source .venv/bin/activate  # Для macOS/Linux
    # или
    .venv\Scripts\activate.bat  # Для Windows CMD
    ```

3.  **Установите зависимости:**
    ```bash
    pip install google-adk
    ```

4.  **Настройте ключ API:**
    Скопируйте файл `.env.example` в `.env`:
    ```bash
    cp .env.example .env
    ```
    Откройте файл `.env` и вставьте ваш ключ API от Google AI Studio:
    ```
    GOOGLE_API_KEY="ВАШ_КЛЮЧ_API"
    ```

## Разработка и тестирование агентов

### Создание нового агента

Вы можете создать нового агента с помощью команды `adk create`:
```bash
adk create src/имя_вашего_агента
```

### Запуск агента

1.  **Через командную строку:**
    ```bash
    adk run src/имя_агента
    ```

2.  **Через веб-интерфейс:**
    Запустите веб-сервер из корневой директории проекта:
    ```bash
    adk web --port 8000
    ```
    После этого откройте в браузере `http://localhost:8000`.

### Тестирование

На данный момент тестирование подразумевает запуск агента и проверку корректности выполнения его задач. В будущем мы добавим более строгие методы тестирования.
